Project NuLens

			NuLens presents a new lens through which information can be viewd. By fostering collaboration among 
			Manufacturing Industry and Supply-Chain partners we seek to usher in the Industry 4.0 Revolution. 
			NuLens will promote effeciency by advocating Right-to-Repair and Data Democratization ideals.
			
			The Goals of Project NuLens are:
			
				1. Implement API for aggregation of product technical data.
				2. Real Time asset tracking of product quality, deficiencies, and repair.
				3. API integration with "Smart"" IoT systems.
				4. Develop learning models for "Smart" networks.
				5. Optimize supply chain distribution.

Clients

			NuLens is geared towards several industry clients:
			
				1. Product Engineers
				2. Product Designers
				3. Origional Equipment Manufacturers
				4. Supply-Chain MAnagers
				5. Distributors
				6. End Users
